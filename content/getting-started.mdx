# Démarrage rapide

Bienvenue sur Grubano ! Ce guide vous permet de configurer votre restaurant et de commencer à recevoir des commandes en **15 minutes**.

<Callout type="info">
  Vous avez besoin de vos identifiants d'accès (email + mot de passe) fournis par votre gestionnaire de compte Grubano.
</Callout>

## Connexion

<Steps>

### Se connecter au dashboard

Rendez-vous sur [grubano.com](https://grubano.com) et saisissez vos identifiants. Selon votre rôle, vous serez redirigé vers votre portail :

- **Restaurateur** → `/dashboard`
- **Franchisé** → `/franchise`
- **Créateur** → `/creators`

### Créer votre première marque

Dans le menu principal, allez dans **Marques** → **Nouvelle marque**.

Renseignez :
- Nom de la marque
- Cuisine (ex. : Italienne, Asiatique)
- Logo et photo de couverture

### Configurer votre menu

Accédez à **Menu** → **Ajouter un plat**.

Utilisez la fonction **Scan IA** pour créer automatiquement une fiche plat à partir d'une photo : l'IA détecte le nom, la description et les allergènes.

### Configurer votre restaurant

Dans **Profil Restaurant**, renseignez :
- Adresse de livraison
- Horaires d'ouverture
- Délai de livraison moyen
- Commande minimum

### Activer votre restaurant

Basculez le statut sur **Actif** pour commencer à recevoir des commandes via l'application consommateur.

</Steps>

## Fonctionnalités essentielles

Après l'onboarding initial, explorez ces fonctionnalités clés :

| Fonctionnalité | Chemin | Description |
|---|---|---|
| Tableau de bord | `/dashboard` | KPIs en temps réel |
| Gestion des commandes | `/orders` | Suivi et statuts |
| Stock | `/stocks` | Inventaire avec alertes DLC |
| Briefing IA | `/briefing` | Résumé matinal personnalisé |
| Analytiques | `/analytics` | Revenus et heures de pointe |
| Programme fidélité | `/loyalty` | Gestion des points clients |
