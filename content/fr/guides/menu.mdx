# Menu & Scan IA

Photographiez votre carte papier : l'IA reconnaît chaque plat, propose un nom, une description appétissante et les allergènes potentiels — vous vérifiez, corrigez si besoin, et publiez votre menu en quelques minutes.

<FlowDiagram steps={["Scanner la carte","Vérifier les plats","Publier","Mettre à jour"]} />

## Ce que ça fait

Saisir une carte complète à la main — noms, descriptions, prix, allergènes — prend des heures et décourage souvent les restaurants qui rejoignent une plateforme. Le scan IA vous fait gagner ce temps : vous prenez une photo de chaque plat ou de votre menu imprimé, l'IA extrait automatiquement le nom en français, rédige une description de une à deux phrases qui donne envie, identifie les ingrédients principaux et détecte les allergènes parmi les quatorze réglementaires (gluten, lactose, œuf, soja, arachide, fruits à coque, poisson, crustacés, mollusques, céleri, moutarde, sésame, sulfites, lupin).

L'IA propose également une fourchette de calories estimée, classe le plat dans la catégorie adaptée (entrées, plats, desserts, boissons) et suggère des labels pertinents (végétarien, halal, sans gluten, épicé). Vous conservez toujours la main : chaque proposition apparaît dans votre [tableau de bord](/fr/guides/restaurant/), où vous pouvez corriger le nom, réécrire la description, ajuster le prix ou compléter les allergènes avant de publier. Une fois validé, le plat rejoint votre menu en ligne et les clients peuvent le commander ; vous pouvez également gérer la disponibilité en temps réel via la page [stocks](/fr/guides/stocks/) si un ingrédient vient à manquer.

## Comment l'utiliser

<Steps>

### Préparez vos photos

Prenez une photo nette de chaque plat (éclairage naturel, cadrage serré sur l'assiette) ou photographiez directement votre carte papier si elle est lisible. Les formats acceptés sont JPEG, PNG, GIF et WebP.

### Envoyez l'image au scan

Dans votre tableau de bord, accédez à la section Menu et cliquez sur « Scanner un plat ». Téléchargez la photo ; l'IA analyse l'image et renvoie en quelques secondes un objet JSON contenant le nom du plat, la description, les ingrédients, les allergènes détectés, la fourchette calorique, la catégorie suggérée et les labels recommandés.

### Vérifiez et corrigez

Relisez chaque champ proposé : corrigez les fautes éventuelles, ajustez la description pour qu'elle reflète votre style, vérifiez que les allergènes correspondent bien à votre recette, ajoutez le prix de vente et éventuellement un prix barré (`comparePrice`) si vous souhaitez afficher une promotion.

### Publiez le plat

Une fois satisfait, enregistrez : l'entrée est créée dans votre menu avec le statut `available: true` par défaut. Le plat apparaît immédiatement sur votre page restaurant, classé par catégorie puis par ordre alphabétique. Vous pouvez à tout moment revenir dans le tableau de bord pour modifier le prix, la disponibilité, les photos ou les options (suppléments, choix de taille).

</Steps>

## Bon à savoir

<Callout type="info">
L'IA détecte uniquement les allergènes présents dans la liste réglementaire des quatorze allergènes majeurs. Si votre plat contient un ingrédient rare ou une préparation spécifique non visible sur la photo, ajoutez manuellement l'allergène correspondant avant de publier pour respecter vos obligations d'information.
</Callout>
