## Le principe (et le pourquoi)

Chaque fois que vous [commandez](/fr/guides/consumer/) sur Grubano, vous gagnez des points de fidélité. Ces points s’accumulent dans votre cagnotte et se convertissent en euros : un solde réel que vous pouvez dépenser lors de votre prochaine commande. Plus vous commandez, plus vous montez de palier (Bronze → Silver → Gold → Platine) et plus votre solde grandit.

Parallèlement, vous disposez d’un code de parrainage unique. Lorsqu’un proche l’utilise pour passer sa première commande, il reçoit une réduction de bienvenue — une façon simple de faire découvrir Grubano autour de vous. Votre cagnotte, vos paliers et votre code sont visibles en permanence dans votre compte.

## Consulter votre cagnotte et votre palier

<Steps>

### Accédez à votre compte

Une fois connecté, rendez-vous dans la rubrique « Mon compte » de l’application. Votre solde de points s’affiche en euros (le montant de crédit dont vous disposez), ainsi que votre palier actuel (Bronze, Silver, Gold ou Platine).

### Suivez votre progression

L’interface vous indique le nombre de points qu’il vous reste à cumuler pour atteindre le palier suivant. Par exemple, si vous êtes Bronze (0–99 points) et que vous avez 75 points, il vous en manque 25 pour passer Silver (100 points).

### Consultez votre historique de points

La section « Historique » détaille tous vos mouvements de points : les points gagnés lors de chaque commande validée, les points dépensés (quand vous utilisez votre crédit) et, le cas échéant, les points recrédités en cas d’annulation. Chaque ligne indique la date, le type de mouvement et l’équivalent en euros.

</Steps>

## Parrainer vos proches

<Steps>

### Récupérez votre code de parrainage

Dans votre compte, votre code de parrainage personnel apparaît sous forme d’une chaîne courte (8 caractères en majuscules). Ce code est unique et permanent.

### Partagez-le

Communiquez ce code à un proche qui n’a jamais commandé sur Grubano. Lorsqu’il passera sa première commande et saisira votre code, il bénéficiera d’une réduction de bienvenue (le montant exact de cette réduction est défini dans la configuration de parrainage en vigueur).

### Le parrainage est automatique

Dès que votre proche valide sa commande avec votre code, le système enregistre le lien de parrainage. Vous n’avez aucune autre action à effectuer : le mécanisme est transparent et l’application garde la trace de la relation.

</Steps>

## Barème de conversion points → euros

Votre solde de points se convertit en euros de crédit selon un taux fixe. Par exemple, avec le barème actuel :

- **100 points** = 5 € de crédit
- **200 points** = 10 € de crédit
- **400 points** = 20 € de crédit

Si vous avez 240 points, vous disposez de 12 € de crédit utilisables sur votre prochaine commande. Le montant exact de votre solde en euros est affiché en temps réel dans votre compte — vous n’avez pas à calculer.

<Callout type="info">
Le taux de conversion est géré par Grubano et s’applique automatiquement. Lorsque vous choisissez d’utiliser votre crédit au moment de payer, le système déduit le montant correspondant de votre panier et ajuste votre solde de points en conséquence.
</Callout>

## Les quatre paliers de fidélité

Votre palier reflète le total de points que vous avez cumulés depuis votre inscription. Les seuils sont les suivants :

- **Bronze** : de 0 à 99 points (palier de départ)
- **Silver** : à partir de 100 points
- **Gold** : à partir de 200 points
- **Platine** : à partir de 400 points

Votre palier monte automatiquement dès que vous franchissez un seuil. Il ne redescend jamais, même si vous dépensez des points : seul le total cumulé compte. Le palier est un indicateur de fidélité visible dans votre compte.

## Tirer le meilleur parti de votre cagnotte

<Callout type="default">
- **Consultez régulièrement votre solde** avant de commander : vous saurez si vous disposez d’un crédit suffisant pour couvrir tout ou partie de votre panier.
- **Partagez votre code de parrainage** avec des personnes qui n’ont jamais utilisé Grubano — le code ne fonctionne que pour une première commande.
- **Vérifiez votre historique de points** si vous constatez un écart : chaque mouvement (gain, dépense, recrédit) y figure avec sa date et son montant.
</Callout>

<Eyebrow icon="error">Erreurs à éviter</Eyebrow>

## Les pièges courants

<Callout type="error">
- **Ne partagez pas votre code à quelqu’un qui a déjà commandé** : la réduction de bienvenue ne s’applique qu’à la toute première commande d’un nouveau client.
- **Ne confondez pas points et euros** : votre solde de points se convertit en crédit euro selon le barème affiché dans votre compte — ne tentez pas de calculer vous-même un taux différent.
- **N’utilisez pas votre propre code de parrainage** : le système détecte et bloque les tentatives d’auto-parrainage (même adresse e-mail).
</Callout>
