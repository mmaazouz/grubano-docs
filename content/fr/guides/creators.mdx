# Créateurs

Proposez des recettes qu'un restaurant partenaire cuisinera sous son enseigne, ou recommandez des adresses à votre communauté : dans les deux cas, vous touchez une commission sur les ventes générées.

<FlowDiagram steps={["Candidater","Recette OU lien","Adoption / partage","Commission sur ventes"]} />

## Ce que ça fait

Le programme créateurs vous ouvre deux casquettes distinctes, cumulables à volonté. **Première casquette : concepteur de recettes.** Vous soumettez des idées de plats — avec nom, description, ingrédients, photo et prix suggéré — que les restaurateurs partenaires découvrent dans leur espace. Lorsqu'un établissement adopte votre recette, il la cuisine dans sa propre cuisine, sous son enseigne, et vous percevez une commission sur chaque vente du plat dans les limites fixées par Grubano. Votre création figure alors dans [le menu](/fr/guides/menu/) du restaurant, enrichie d'un badge portant votre nom et votre photo ; les clients qui [commandent](/fr/guides/consumer/) ce plat contribuent directement à vos revenus. **Seconde casquette : recommandation par lien d'affiliation.** Vous recevez un code et un lien personnalisés que vous partagez sur vos réseaux ; chaque nouveau client qui passe commande via ce lien déclenche une commission dans les limites fixées par Grubano. Les deux flux sont indépendants — vous pouvez vous concentrer sur les recettes, sur l'affiliation, ou mener les deux de front — et se pilotent depuis un tableau de bord unifié qui affiche vos gains, vos plats adoptés et vos parrainages actifs.

## Comment l'utiliser

<Steps>

### Candidater au programme

Rendez-vous sur la page publique Créateurs et cliquez sur **Rejoindre le programme**. Remplissez le formulaire avec votre nom, email, biographie, liens vers vos réseaux sociaux (Instagram, TikTok, YouTube) et le nombre d'abonnés. Ajoutez entre une et trois idées de plats — nom, description, type de cuisine — pour illustrer votre style culinaire.

### Vérifier votre chaîne YouTube (optionnel)

Si vous avez fourni un lien YouTube, l'application génère un code de vérification au format `GRUBANO-XXXXXX` que vous devez coller dans la description de votre chaîne. Cliquez ensuite sur **Vérifier ma chaîne** : le système lit la description, contrôle la présence du code et analyse le contenu récent de vos vidéos pour confirmer la pertinence culinaire. Une vérification réussie active le badge vérifié sur votre profil et facilite l'adoption de vos recettes par les restaurateurs.

### Soumettre une recette

Depuis votre espace créateur, accédez à **Mes recettes** puis **Soumettre un plat**. Renseignez le nom, la description détaillée, la liste des ingrédients, le type de cuisine, un prix suggéré et téléchargez une photo attrayante. La recette passe au statut `pending` le temps que l'équipe Grubano vérifie la qualité et la faisabilité ; une fois approuvée (`approved`), elle devient visible dans l'espace des restaurateurs qui peuvent l'adopter.

### Partager votre lien d'affiliation

Dans **Mon affiliation**, copiez votre code personnel et le lien associé. Partagez-les sur vos réseaux, dans vos vidéos ou vos stories : chaque nouveau client qui commande via ce lien active une commission sur sa première commande. Le tableau **Clients parrainés** liste les parrainages actifs et les gains correspondants.

### Suivre vos revenus

La page **Mes revenus** sépare les deux sources : commission sur les recettes adoptées d'un côté, commission d'affiliation de l'autre. Un graphique sur trente jours glissants affiche l'évolution de vos gains totaux ; des compteurs synthétisent le nombre de ventes par plat et le chiffre d'affaires généré par vos parrainages.

</Steps>

## Bon à savoir

<Callout type="info">
La commission que vous percevez sur chaque vente — recette ou affiliation — est définie avec vous au moment de l'ouverture de votre compte créateur et reste stable tant que le contrat n'est pas renégocié. Le badge vérifié améliore la confiance des restaurateurs et accélère l'adoption de vos plats, mais n'est pas obligatoire pour soumettre des recettes ou partager votre lien d'affiliation.
</Callout>
