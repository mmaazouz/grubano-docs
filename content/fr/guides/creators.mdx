# Créateurs

Proposez des recettes que des restaurants adopteront sous leur enseigne, ou recommandez des adresses à votre communauté : vous touchez une commission sur chaque vente générée.

<FlowDiagram steps={["Candidater","Recette OU lien","Adoption / partage","Commission sur ventes"]} />

## Ce que ça fait

Grubano vous offre deux modes de monétisation complémentaires. Le premier consiste à déposer une recette originale : un restaurant partenaire qui l'approuve la cuisine dans sa propre cuisine, sous sa propre marque, et la met en vente sur son [menu](/fr/guides/menu/) ; chaque fois qu'un client [commande](/fr/guides/consumer/) ce plat, vous percevez une commission sur le prix de vente, dans les limites fixées par Grubano. Le second mode repose sur l'affiliation : vous partagez un lien personnalisé vers un restaurant que vous recommandez ; tout nouveau client qui passe commande via ce lien bénéficie d'une réduction de bienvenue, et vous touchez une commission récurrente sur ses achats futurs, toujours dans les limites fixées par Grubano.

Aucun minimum de followers n'est requis pour candidater, mais vous devrez prouver que vous contrôlez bien le compte social que vous déclarez en y publiant un code de vérification fourni après dépôt de candidature. Une fois validé, vous accédez à un tableau de bord qui suit en temps réel le nombre de plats adoptés, les ventes générées et vos gains cumulés. Le classement public des créateurs met en avant ceux qui totalisent le plus de ventes, créant une émulation saine et une visibilité naturelle pour les meilleurs contributeurs.

## Comment l'utiliser

<Steps>

### Déposez votre candidature

Renseignez votre nom, e‑mail, biographie, liens de réseaux sociaux (Instagram, TikTok, YouTube) et nombre approximatif de followers. Ajoutez entre une et trois idées de plat (nom, description, type de cuisine) pour illustrer votre univers culinaire.

### Prouvez que vous contrôlez votre chaîne

Après validation du formulaire, vous recevez un code de vérification unique. Publiez ce code dans la description de votre chaîne YouTube (ou le réseau social déclaré) pour confirmer la propriété du compte.

### Soumettez des recettes détaillées

Une fois vérifié, connectez‑vous pour créer des fiches de plat complètes : nom, description longue, liste d'ingrédients, type de cuisine, prix suggéré et photo. Chaque recette passe en statut `pending` jusqu'à ce qu'un restaurateur l'approuve.

### Ou partagez votre lien d'affiliation

Grubano génère automatiquement un code de parrainage (par exemple `MARCO20`) et une URL courte (`/r/votre‑slug`). Diffusez ce lien à votre communauté : le premier achat d'un nouveau client lui accorde une réduction de bienvenue, et vous commencez à percevoir une commission sur ses commandes, dans les limites fixées par Grubano.

### Suivez vos gains

Votre tableau de bord regroupe le nombre de recettes adoptées, le total de ventes réalisées, vos revenus cumulés et votre position dans le classement public des créateurs.

</Steps>

## Bon à savoir

<Callout type="info">
Une recette adoptée par un restaurant reste visible sous l'enseigne de ce restaurant, jamais sous votre propre nom côté client final : c'est le restaurateur qui engage sa réputation et sa marque. Vous apparaissez uniquement dans le classement public des créateurs et dans les rapports internes du restaurateur qui a choisi votre recette.
</Callout>
