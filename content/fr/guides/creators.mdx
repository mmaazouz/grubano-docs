## Le principe (et le pourquoi)

Vous êtes chef, passionné de cuisine, et vous avez des idées de plats originaux. Plutôt que d’ouvrir votre propre restaurant, vous publiez vos recettes sur Grubano. Un restaurateur partenaire les découvre, les adopte, et les cuisine dans **sa** cuisine, sous **son** enseigne. Vous restez créateur, sans gérer ni stock ni logistique.

À chaque commande de votre plat, vous touchez 2 % du prix de vente (HT). Le restaurateur gère le [menu](/fr/guides/menu/), la préparation, et la relation client ; vous suivez vos gains et le nombre de ventes en temps réel.

## Publier une recette et toucher vos gains

<Steps>

### Candidater au programme

Remplissez le formulaire de candidature : nom, email, bio, réseaux sociaux (Instagram, TikTok, YouTube), et jusqu’à 3 concepts de plats (nom, description, type de cuisine). Si vous fournissez un lien YouTube, collez le code de vérification fourni dans la description de votre chaîne pour obtenir le badge vérifié.

### Soumettre une recette

Une fois votre profil créé, publiez vos recettes détaillées : nom, description, ingrédients, type de cuisine, prix suggéré, photo. Vous pouvez enregistrer un brouillon (privé) ou soumettre directement. Les recettes soumises passent par une vérification automatique : approuvée, en attente de révision, ou refusée (avec motif).

### Adoption par un restaurant

Les restaurateurs parcourent le catalogue des recettes validées et adoptent celles qui les intéressent. Une fois adoptée, votre recette apparaît dans le menu du restaurant, sous sa marque. Vous n’intervenez pas dans la préparation ni la vente.

### Suivre vos revenus

Chaque commande du plat génère une vente enregistrée. Vous touchez 2 % du prix de vente (HT) par assiette vendue. Le tableau de bord créateur affiche le nombre de ventes par recette, le total des gains, et la liste des restaurants qui cuisinent vos plats.

</Steps>

## Maximiser vos chances d’adoption

<Callout type="default">
- **Photo appétissante** : une image professionnelle multiplie par trois les chances d’adoption.
- **Description claire** : ingrédients précis, étapes de préparation, temps estimé — le restaurateur doit pouvoir reproduire le plat facilement.
- **Prix cohérent** : un prix suggéré réaliste (aligné sur le type de cuisine et le marché local) rassure le restaurateur.
- **Profil complet** : bio, réseaux sociaux, badge vérifié si vous avez une audience YouTube — les restaurateurs privilégient les créateurs crédibles.
</Callout>

<Eyebrow icon="error">Erreurs à éviter</Eyebrow>

## Les pièges courants

<Callout type="error">
- **Recette trop complexe** : un plat qui nécessite 15 ingrédients rares ou 45 minutes de cuisson sera rarement adopté ; privilégiez l’efficacité opérationnelle.
- **Manque de détails** : une description vague (« salade fraîche ») sans liste d’ingrédients sera refusée ou ignorée par les restaurateurs.
- **Prix irréaliste** : suggérer 25 € pour une salade simple ou 5 € pour un plat élaboré dessert votre crédibilité ; alignez-vous sur le marché.
- **Plagiat** : soumettre une recette qui existe déjà ailleurs (autre créateur, restaurant connu) entraîne un refus automatique.
</Callout>
