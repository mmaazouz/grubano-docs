# Créateurs

Proposez vos recettes aux restaurants partenaires ou recommandez des adresses à votre communauté : dans les deux cas, vous touchez une commission sur les ventes générées.

<FlowDiagram steps={["Candidater","Recette OU lien","Adoption / partage","Commission sur ventes"]} />

## Ce que ça fait

Le programme Créateurs ouvre deux parcours distincts et cumulables, selon votre profil et vos objectifs.

**Chef créateur** : vous concevez une recette — plat complet, dessert, spécialité — que les restaurants partenaires peuvent adopter et cuisiner sous leur propre enseigne. Vous conservez la propriété intellectuelle de la création ; le restaurateur bénéficie d'un concept clé en main qu'il n'aurait pas eu le temps ni les ressources de développer seul. Chaque vente de votre recette vous rapporte une commission dans les limites fixées par Grubano. Votre [menu](/fr/guides/menu/) s'enrichit d'une dimension collaborative : au lieu de tout créer en interne, vous pouvez piocher dans le catalogue de recettes validées, tester une nouvelle cuisine sans embaucher un spécialiste, et diversifier votre offre à moindre risque.

**Influenceur affilié** : vous générez un lien personnalisé que vous partagez sur vos réseaux (Instagram, TikTok, YouTube). Vos abonnés découvrent les restaurants locaux que vous recommandez ; lorsqu'ils [commandent](/fr/guides/consumer/) via votre lien, vous touchez une commission sur la transaction. Pas besoin de cuisiner ni de gérer de stock : vous monétisez votre audience en orientant vos followers vers des adresses de qualité, pendant que les restaurateurs gagnent en visibilité auprès d'une communauté qualifiée.

Les deux rôles se cumulent librement : un chef peut aussi partager son lien d'affiliation, un influenceur peut proposer une recette s'il maîtrise la cuisine. La candidature unique débloque les deux parcours ; vous choisissez ensuite celui — ou ceux — que vous activez.

## Comment l'utiliser

<Steps>

### Candidater au programme

Rendez-vous sur la page Créateurs et remplissez le formulaire de candidature : nom, bio courte, liens vers vos réseaux sociaux (Instagram, TikTok, YouTube), nombre d'abonnés. Si vous souhaitez proposer des recettes (rôle Chef), décrivez jusqu'à trois concepts de plats — nom, description, type de cuisine — pour illustrer votre vision. Si vous visez uniquement l'affiliation (rôle Influenceur), la description de plats est optionnelle.

L'équipe Grubano examine chaque dossier manuellement. Un e-mail de validation vous parviendra si votre profil correspond aux critères de qualité et d'authenticité de la plateforme.

### Vérifier la propriété de vos comptes

Pour éviter les usurpations d'identité, vous devez prouver que vous contrôlez bien les comptes sociaux déclarés. Un code de vérification unique vous est fourni après la candidature : copiez-le et collez-le dans la description de votre chaîne YouTube (ou le champ bio de votre profil Instagram / TikTok, selon les instructions reçues). Une fois le code publié, cliquez sur « Vérifier » : le système contrôle automatiquement sa présence et active votre compte Créateur.

### Choisir votre casquette

Une fois vérifié, vous accédez à votre tableau de bord Créateur. Activez le rôle **Chef** pour accéder à l'éditeur de recettes, ou le rôle **Influenceur** pour générer votre lien d'affiliation personnalisé. Vous pouvez cocher les deux si vous le souhaitez : chaque rôle fonctionne de manière indépendante.

### Soumettre une recette (Chef) ou partager votre lien (Influenceur)

**Parcours Chef** : dans l'éditeur, créez votre recette en renseignant le nom, la description, la liste des ingrédients, le type de cuisine, et un prix suggéré. Téléchargez une photo attractive. Vous pouvez enregistrer un brouillon (visible uniquement par vous) ou soumettre la recette pour validation : un contrôle automatisé vérifie la cohérence du contenu (pas de spam, description suffisante, tarif réaliste). Si tout est conforme, la recette passe immédiatement au statut « approuvée » et devient visible dans le catalogue des restaurateurs. En cas de doute, elle reste en attente de relecture manuelle. Une fois approuvée, les restaurants peuvent l'adopter ; chaque vente génère votre commission.

**Parcours Influenceur** : récupérez votre lien d'affiliation unique (de la forme `grubano.fr/r/votrecode`) depuis le tableau de bord. Partagez-le dans vos stories, vos vidéos, vos posts. Lorsqu'un utilisateur clique et commande dans les jours qui suivent, le système enregistre l'attribution : vous touchez une commission sur la transaction, dans les limites fixées par Grubano.

### Suivre vos performances

Le tableau de bord Créateur affiche, pour chaque rôle, les indicateurs clés : nombre de recettes adoptées et total des ventes pour les chefs, nombre de clics et de conversions pour les influenceurs. Vos gains cumulés apparaissent dans la colonne « Revenus totaux ». Un classement public (leaderboard) met en avant les créateurs les plus performants de la plateforme, tous rôles confondus.

</Steps>

## Bon à savoir

<Callout type="info">
La candidature est unique, mais les rôles Chef et Influenceur fonctionnent de manière totalement indépendante : vous pouvez en activer un seul, ou les deux simultanément. Un code d'affiliation vous est attribué dès la vérification ; il reste actif même si vous désactivez temporairement le rôle Influenceur. De même, vos recettes approuvées restent visibles dans le catalogue tant que vous conservez le rôle Chef, même si vous ne soumettez plus de nouveaux plats.
</Callout>
