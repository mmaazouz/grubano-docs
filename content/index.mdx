# Bienvenue sur Grubano Docs

**Grubano** est une plateforme de gestion de dark kitchens multi-marques. Ce site documente l'ensemble des fonctionnalités pour les restaurateurs, consommateurs, franchisés et créateurs.

## Portails disponibles

| Portail | Accès | Description |
|---------|-------|-------------|
| **Dashboard** | `grubano.com/dashboard` | Gestion opérationnelle du restaurant |
| **Consumer** | `grubano.com/eat` | Application mobile de commande |
| **Franchise** | `grubano.com/franchise` | Portail réseau franchisés |
| **Créateurs** | `grubano.com/creators` | Soumission de plats et commissions |
